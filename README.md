# Ounces-Optimizer
Ounces achievement optimizer for UMA Obuasi underground mine — truck allocation tool for daily production planning.
What it does
Calculates the best truck allocation across active ore stopes and development headings to hit the daily ounces target. Enter your target, plan tonnes, and active stopes — the optimizer finds the top 3 combinations instantly.
How to use
Open the link in any browser on phone, tablet, or computer. No login or installation needed.
